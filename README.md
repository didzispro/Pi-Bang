# Pi-Bang

🏓 Pi Bang
My first Unity 2D Pong game!

🎮 Features
Classic Pong gameplay
Glow effects and particle effects
Ball trail
Screen shake on hits
Power-ups (bigger paddle, 2x balls)
Difficulty settings (Easy, Medium, Hard)
2-player mode
Pause menu
Settings with volume control

Screenshot%20MainMenu.png

🎯 Controls
Player 1:

W = Move Up
S = Move Down
Player 2 (2-Player Mode):

Arrow Up = Move Up
Arrow Down = Move Down
Pause:

ESC = Pause/Resume
🛠️ Built With
Unity 2022.x
C#
Universal Render Pipeline (URP)
